Project Name: Record Management System (Java Swing & MySQL)

Description:
A desktop-based Record Management System built with Java Swing for GUI and MySQL as the backend database. The system allows users to add, update, delete, search, and view records such as name, phone, email, and address. It includes admin login authentication and duplicate prevention for phone and email fields.

Key Features:

Admin Login: Secure authentication using username and password.

CRUD Operations: Create, Read, Update, Delete records in a user-friendly interface.

Duplicate Check: Prevents duplicate phone numbers or emails for integrity.

Search & View All: Quick search by name and view all records in a table.

User Interface: Built with Java Swing, using JTable to display records dynamically.

Database: Uses MySQL for data storage; can be imported via .sql file.

Technologies Used:

Java (Swing, JDBC)

MySQL

PreparedStatement for secure database operations
